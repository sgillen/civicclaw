What is this?

CivicClaw is powered by sf-civic-digest, an open-source data skill that collects information from 10+ government sources and 5 local newsrooms. Scripts run weekly, an AI agent writes the narrative, and the result lands here.

No ads, no paywall, no account required. Just civic data, clearly presented.

Data sources: Legistar, SFMTA, SF Planning, Socrata, SF Ethics, Mission Local, 48 Hills, SF Standard, Streetsblog SF, SF YIMBY
