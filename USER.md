# USER.md — Civic Digest Profile

This file is a compatibility shim.

Canonical per-user profiles live under `users/<name>/USER.md`.

- Start from `users/TEMPLATE.md`
- Keep personalized user profiles out of git if they contain private preferences
- Use this root file only as a generic fallback when no user-specific profile is selected

If this file is missing or left generic, reports default to neutral tone with no filtering preferences.
